JavaScript

There are some scripts connected to the page. Their files can be found in “js” folder.

  1. jQuery;
  2. easing;
  3. holder;
  4. jquery.fullpage;
  5. hoverintent;
  6. superfish;
  7. select2;
  8. jplayer;
  9. collapse, popover, tab, transition;
  10. and custom js file

Here is a brief descriptions of the scripts on the page:

jQuery is a JavaScript library which simplifies code writing with JavaScript and allows to connect other plugins.

Easing is a plugin used to create non-linear time-functions.

Holder is a image placeholders

Jquery.fullpage is a jQuery Plugin for fullscreen scrolling websites

Hoverintent is a plug-in that attempts to determine the user's intent

Superfish is a jQuery plugin that adds usability enhancements to existing multi-level drop-down menus

Select2 is the jQuery replacement for select boxes

 

Jpayer is the jQuery HTML5 Audio / Video Library

 

Collapse, popover, tab, transition - JavaScript Bootstrap's components

 

Custom scripts.js file are used to initialize other scripts and it initialize all the animation.

Slider Revolution

Slider Revolution is an innovative, responsive jQuery Slider Plugin that displays your content the beautiful way.

Whether it's a Slider, Carousel, Hero Scene or even a whole Front Page, you will be telling your own stories in no time!

Slider Revolution also requeires a modern browser with full Javascript and HTML5 support.
